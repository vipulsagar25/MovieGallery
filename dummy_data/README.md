# Dummy Data Directory

This directory contains `data.json`, which holds the mock responses and datasets previously used for frontend scaffolding and disconnected local testing. 

## Purpose
It allows the frontend component to be populated and tested before a complete backend or database layer is connected. 

## Usage in Production
These files **do not affect the production deployment** and are completely excluded from the worker or API build processes. They exist solely for historical context and isolated UI development.
